Classes

The classes for applications are worth 9 points. The classes for design are worth 6 points.

Applications

We will solve 3 problems, each in several languages:

  • Kent Weeks 14-15: Java or Python
  • Kent Weeks 15-16: Rust or Scala
  • Kent Weeks 16-17: Haskell or OCaml

The problems are:

  • multiply two numbers
  • convert an expression from infix notation to postfix notation
  • topological sort

The marking is done on SPOJ. (You will need an account) You get 1 point for each reasonable attempt at solving a problem. (3 problems x 3 weeks = 9 points) In class, you can work in your favorite development environment, if it is available on the university's computers. Alternatively, you can write your code on ideone.
