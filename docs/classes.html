Classes

The classes for applications are worth 9 points. The classes for design are worth 6 points.

Applications

We will solve 3 problems, each in several languages:

  • Kent Weeks 14-15: Java or Python
  • Kent Weeks 15-16: Rust or Scala
  • Kent Weeks 16-17: Haskell or OCaml

The problems are:

  • multiply two numbers
  • convert an expression from infix notation to postfix notation
  • lexicographical-minimum topological-sort

The marking is done on SPOJ. You will need an account. Please use your Kent email address. (3 problems x 3 sets = 9 points) In class, you can work in your favorite development environment, if it is available on the university's computers. Alternatively, you can write your code on ideone.

Hints:

  1. For multiplication, many languages know how to do it. For those that don't, see Multiplication Algorithm in Wikipedia; it's the same as what you do on paper.
  2. For converting infix to postfix, see Recursive Descent Parser in Wikipedia.
  3. For topological-sort, follow the links in the problem itself, again to Wikipedia. Note that this problem handles a special case of the programming assignment.
