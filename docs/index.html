Programming Languages:
Applications and Design

This module has two halves.

  1. Applications: we solve three problems in three languages, and we discuss how different solutions to the same problem compare.
  2. Design: we cover basics of programming language theory, building on the examples from the first half

The assessment has three components.

  1. Classes: you will solve three problems, each in three out of the five languages we study
  2. Programming Homework: you will implement a dependency manager
  3. Paper Presentation: you will prepare a presentation summarising a research paper related to programming languages, and you will be assessed in a viva
